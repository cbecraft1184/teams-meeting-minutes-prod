# Technology Stack Summary

The solution is a full-stack TypeScript application with a React frontend and Node.js/Express backend, using PostgreSQL for data storage and Drizzle ORM for database operations. Microsoft Graph API handles Teams integration through webhook subscriptions, while Azure OpenAI Service provides AI capabilities via GPT-4o for minutes generation and Whisper for transcription. The entire stack runs on Azure Commercial Cloud with Azure Active Directory for authentication and role-based access control.
